    Class ScrollBarContents

    スクロールバーで操作するコンテンツ

    コンテンツのサイズを変更した場合"changed_contents_size"イベントをemitすると、スクロールバーが再描画される。

    • getDisplayRate(isHorizontal: boolean): number
